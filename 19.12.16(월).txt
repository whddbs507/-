< Literal > : 자바 언어에서 값을 표기하는 문법
10, 10L, 3.14f, 3.14d, false, true
65, 0x41, 'A', '\u0041', "ABC"

< Variable >
- primitive data type
정수 : byte, short, int, long
부동소수점 : float, double
논리 : boolean
문자 : char
- referance ( 그 외 나머지 (Date, String, Scanner 등..)

< Array >
int[] arr = new int[3]; << arr은 레퍼런스. new int[3]은 인스턴스
배열의 레퍼런스 : 주소 값
배열의 인스턴스 : 값의 예
arr2 = arr; (arr의 주소값을 arr2에 대입하라)
배열은 주소가 없으면 접근이 절대 불가
